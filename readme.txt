A plugin for displaying sticky posts based on custom taxonomy, order, and much more.

== Description ==

Gives you a control for displaying your sticky post easily with many options.

<h4>Key Features & Options</h4>
<ul>
<li>No additional admin panel</li>
<li>Post ordering</li>
<li>Total posts</li>
<li>Post excerpt  and thumbnail option</li>
<li>Date and comment options</li>
<li>Widget intro and outro-text for your additional information, support HTML</li>
</ul>

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
1. Go to the menu 'Plugins' -> 'Install' and search for 'Sticky Post'
2. Click 'install'.
